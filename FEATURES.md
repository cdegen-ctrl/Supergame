## Описание фичей

### 1. Мобильная поддержка
- Виртуальные кнопки (←, →, ↑) поверх canvas
- Touch события для всех кнопок
- Поддержка свайпа для прыжка

### 2. Адаптивный canvas
- Canvas масштабируется под размер окна/экрана
- Сохраняет пропорции 8:5 (800x500)
- Пересчёт при resize

### 3. Анимация смерти врага
- После стомпа враг "сплющивается" и отскакивает
- Частицы при смерти

### 4. Комбо-система
- Множитель x2, x3, x4 за стомпы подряд без касания земли
- Визуальный индикатор комбо на экране
- Сброс при касании земли

### 5. Сохранение рекорда
- Сохранение highscore в localStorage
- Сохранение между сессиями (сейчас сбрасывается при перезагрузке)

### 6. Разные типы врагов
- Быстрый Марио (красный оттенок, скорость x2)
- Прыгающий Марио (периодически прыгает)

### 7. Параллакс фона
- Три слоя: дальние горы (медленно), облака (средне), холмы (быстро)
- Плавное движение при перемещении игрока

### 8. Level Select
- Экран выбора уровня после главного меню
- Разблокировка уровней по мере прохождения

| 9 | Монеты на уровнях — подбирай для доп. очков | ✅ Выполнено |

| 11 | Анимация игрока при прыжке (растягивание/сжатие) | ✅ Выполнено |

### 9. Монеты
- Золотые монеты расставлены на каждом уровне
- Подбираются при касании +50 очков каждая
- Небольшая анимация и звук при подборе

### 10. Бонус времени
- Таймер уровня на экране
- Чем быстрее завершил — тем больше бонусных очков

### 11. Squash & Stretch для игрока
- Сжатие при приземлении (Y -20%, X +20%)
- Вытягивание при прыжке (Y +20%, X -20%)
- Наклон при движении влево/вправо

### 12. Улучшенный экран паузы
- Кнопка "Рестарт" 
- Кнопка "В меню"
- Отображение текущего счёта

| # | Фича | Статус |

| 13 | Кнопка мута звука (клавиша V, иконка в HUD) | ✅ Выполнено |

| 17 | Power-up: звезда неуязвимости | ✅ Выполнено |
| 18 | Уникальная тема фона для каждого уровня | ✅ Выполнено |

### 13. Мут звука
- Клавиша V — toggle звука
- Иконка 🔊/🔇 в HUD (правый верхний угол)
- Состояние сохраняется во время сессии

### 14. Двойной прыжок
- В воздухе можно прыгнуть ещё один раз
- Второй прыжок немного слабее первого
- Визуальный эффект: маленькие частицы от второго прыжка
- Сброс при приземлении

### 15. Движущиеся платформы
- Новый класс MovingPlatform — ездит вправо/влево по заданному диапазону
- Платформы на уровнях 3–5 частично заменены движущимися
- Игрок "едет" на платформе, стоя на ней

### 16. Таблица рекордов топ-5
- При завершении игры (GAME_OVER) сравнивает счёт с топ-5
- Сохранение в localStorage (JSON-массив)
- Отображение топ-5 на экране GAME_OVER
- Подсветка нового личного рекорда

### 17. Power-up: звезда
- Золотая мигающая звезда на уровне (фиксированные позиции)
- Подбор даёт 5 сек неуязвимости + убивает врагов при касании
- Визуальный эффект: радужное свечение вокруг игрока
- Звук при подборе

### 18. Темы фона по уровням
- Уровень 1: Стандартный синий день
- Уровень 2: Закат (оранжевое небо)
- Уровень 3: Ночь (тёмно-синее небо + луна)
- Уровень 4: Гроза (серые тучи + молния)
- Уровень 5: Вулкан (тёмно-красное небо + лава)

### 19. Батутные платформы
- Жёлто-оранжевые пружинящие платформы в пропастях
- Приземление на батут запускает игрока высоко вверх (1.7x прыжок)
- Анимация сжатия при срабатывании
- Марио тоже прыгают на батутах
- Сброс двойного прыжка при отскоке

### 20. Марио со щитом
- Новый тип врага: shielded Mario
- Первый стомп снимает щит (синие искры, звук "clang")
- Второй стомп убивает (обычное поведение)
- Визуальный эффект: пульсирующее синее кольцо
- Звёздная неуязвимость обходит щит

### 21. Жизнь за очки
- Каждые 1000 очков набираемых за уровень = +1 жизнь
- Звук и анимация "1UP!"
- Максимум 5 жизней

### 22. Power-up ускорения
- Красный гриб (ирония) на уровнях
- 5 секунд удвоенной скорости + визуальный трейл
- Отображение таймера в HUD

### 23. Эффект погоды
- Капли дождя на тёмных/грозовых уровнях (уровень 4 — гроза)
- Снег на ночных уровнях (уровень 3)
- Анимированные частицы, не мешающие геймплею

### 24. Летающий Марио
- Новый тип 'flying' — движется по синусоидальной волне в воздухе
- Игнорирует гравитацию
- Убивается стомпом сверху (нужно запрыгнуть выше него)
- Значок крылышек ✈ над головой

### 25. Щит для игрока
- Голубая сфера появляется на некоторых уровнях
- Подбирается при касании → поглощает следующий удар врага
- Визуально: синий пульсирующий пузырь вокруг гриба
- Иконка щита в HUD (рядом с жизнями)

### 26. Босс-уровень
- Уровень 8: «Замок» — King Mario с полоской здоровья (3 хп)
- Каждый стомп = -1 хп; становится быстрее с каждым попаданием
- Полоска здоровья над боссом
- Победный экран с конфетти после победы над боссом
