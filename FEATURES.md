## Описание фичей

### 1. Мобильная поддержка
- Виртуальные кнопки (←, →, ↑) поверх canvas
- Touch события для всех кнопок
- Поддержка свайпа для прыжка

### 2. Адаптивный canvas
- Canvas масштабируется под размер окна/экрана
- Сохраняет пропорции 8:5 (800x500)
- Пересчёт при resize

### 3. Анимация смерти врага
- После стомпа враг "сплющивается" и отскакивает
- Частицы при смерти

### 4. Комбо-система
- Множитель x2, x3, x4 за стомпы подряд без касания земли
- Визуальный индикатор комбо на экране
- Сброс при касании земли

### 5. Сохранение рекорда
- Сохранение highscore в localStorage
- Сохранение между сессиями (сейчас сбрасывается при перезагрузке)

### 6. Разные типы врагов
- Быстрый Марио (красный оттенок, скорость x2)
- Прыгающий Марио (периодически прыгает)

### 7. Параллакс фона
- Три слоя: дальние горы (медленно), облака (средне), холмы (быстро)
- Плавное движение при перемещении игрока

### 8. Level Select
- Экран выбора уровня после главного меню
- Разблокировка уровней по мере прохождения
