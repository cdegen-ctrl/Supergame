## Описание фичей

### 1. Мобильная поддержка
- Виртуальные кнопки (←, →, ↑) поверх canvas
- Touch события для всех кнопок
- Поддержка свайпа для прыжка

### 2. Адаптивный canvas
- Canvas масштабируется под размер окна/экрана
- Сохраняет пропорции 8:5 (800x500)
- Пересчёт при resize

### 3. Анимация смерти врага
- После стомпа враг "сплющивается" и отскакивает
- Частицы при смерти

### 4. Комбо-система
- Множитель x2, x3, x4 за стомпы подряд без касания земли
- Визуальный индикатор комбо на экране
- Сброс при касании земли

### 5. Сохранение рекорда
- Сохранение highscore в localStorage
- Сохранение между сессиями (сейчас сбрасывается при перезагрузке)

### 6. Разные типы врагов
- Быстрый Марио (красный оттенок, скорость x2)
- Прыгающий Марио (периодически прыгает)

### 7. Параллакс фона
- Три слоя: дальние горы (медленно), облака (средне), холмы (быстро)
- Плавное движение при перемещении игрока

### 8. Level Select
- Экран выбора уровня после главного меню
- Разблокировка уровней по мере прохождения

### 13. Mute Toggle
- Иконка динамика в HUD (правый нижний угол)
- Клавиша M в игре (не в паузе) отключает/включает звук
- Клик по иконке мышью/тапом на мобильном

### 14. Двойной прыжок
- Игрок может прыгнуть ещё раз в воздухе (второй прыжок чуть слабее)
- Сбрасывается при приземлении
- Визуальный эффект: вспышка при двойном прыжке

### 15. Движущиеся платформы
- Новые платформы на уровнях 4-7 двигаются туда-сюда
- Игрок "стоит" на них и едет вместе
- Враги тоже взаимодействуют с движущимися платформами

### 16. Звезда неуязвимости
- Золотая мигающая звезда на некоторых уровнях
- Подобрать — 10 секунд неуязвимости
- Пока активна — убивает врагов при боковом касании
- Визуальный эффект: радужное свечение вокруг игрока

### 17. Таблица рекордов
- Хранит топ-3 результатов в localStorage
- Показывается на экране Game Over и в главном меню
- Поддержка инициалов игрока (3 буквы)

### 18. Новые уровни (6 и 7)
- Уровень 6: «Небо» — много платформ на разной высоте, быстрые враги
- Уровень 7: «Хаос» — движущиеся платформы + все типы врагов

### 19. Погодные эффекты
- Уровни 5-6: падающий дождь (наклонные капли)
- Уровень 7: снегопад (кружащиеся снежинки)
- Чисто декоративные, не мешают геймплею

### 20. Монеты из врагов
- При стомпе Марио с вероятностью 50% выпадает монета
- Монета летит вверх по дуге, затем приземляется и становится подбираемой
- Даёт дополнительные +50 очков при подборе

### 21. Щит (Shield)
- Синий пауэр-ап на некоторых уровнях
- При подборе игрок получает защиту от одного бокового удара
- Визуальный эффект: синее свечение вокруг игрока
- При поглощении удара: анимация разбитого щита

### 22. Броне-Марио
- Новый тип врага с серым шлемом
- Первый стомп: снимает броню (враг выживает, игрок отскакивает)
- Второй стомп: убивает
- Появляется на уровнях 4-7

### 23. Система достижений
- Всплывающие уведомления в верхней части экрана (3 сек)
- Достижения: «Первый стомп», «Звёздная мощь», «Комбо-мастер x5», «Коллекционер (10 монет)», «Без потерь (уровень без смертей)»

### 24. Анимация перехода между уровнями
- Новое состояние LEVEL_TRANSITION
- Чёрные полосы «задёргиваются» к центру (закрытие), показывается «УРОВЕНЬ X», затем открываются
- Длительность: ~1.5 секунды

### 29. Сердечки жизней
- Вместо текста «LIVES: X» показывать иконки сердечек ❤️
- Заполненное сердце — активная жизнь, пустое — потерянная
- Максимум 3 сердца, поддержка любого количества жизней

### 30. Power-up Ускорение
- Зелёный пауэр-ап с молнией ⚡ на некоторых уровнях
- При подборе: скорость движения x2 на 5 секунд
- Полоска таймера в HUD (зелёная, под звездой)
- Визуальный зелёный след при движении

### 31. Уровень 8 «Кошмар»
- Самый сложный уровень, все типы платформ и врагов
- Минимальные опорные точки, максимальное количество движущихся платформ
- Много врагов, высокая скорость, мало места для манёвра
- Доступен после прохождения уровня 7

### 32. Анимированный счётчик
- Очки в HUD не появляются мгновенно, а «подъезжают» плавно
- displayScore постепенно догоняет реальное значение
- Создаёт ощущение живого интерфейса

### 33. Кастомизация цвета
- На экране Level Select можно выбрать цвет шляпки гриба
- Варианты: красный (по умолчанию), синий, зелёный, жёлтый, фиолетовый
- Выбор сохраняется в localStorage

### 34. Детальная статистика Game Over
- Показывает: убито врагов, монет собрано, максимальное комбо, пройдено уровней
- Обновляется в реальном времени во время игры
- Красивая таблица на экране Game Over

### 35. Прыжок от стены

- Прыжок от стены: нажать прыжок при скольжении — отталкиваешься в сторону
- Пыль при скольжении, золотой контур сбоку
- Индикатор в HUD «↑ ПРЫЖОК ОТ СТЕНЫ!»

### 36. Магнит монет
- Розовый power-up 🧲 на уровнях 3-8
- При подборе: все монеты в радиусе 180px летят к игроку 7 сек
- Полоска таймера в HUD
- Позволяет собирать монеты без захода в труднодоступные места

### 37. Фоновая музыка
- Процедурная chiptune музыка через Web Audio API
- Разные мелодии для дня (1-4), сумерек (5-6) и ночи (7-8)
- Синхронизирована с кнопкой mute / клавишей M
- Плавный переход между уровнями

### 38. Уровень 9 «Босс»
- Новый уровень: простая арена, финальный бой
- BossMarco — гигантский Марио (3x размер), 5 хитов
- Способности: топать ногой (волна), ускорение при низком HP
- Полоска HP у босса вверху экрана
- После победы — экран «ТЫ ПОБЕДИЛ!»

### 39. Контрольные точки
- Флаг-чекпоинт посередине длинных уровней (5-9)
- При касании: флаг анимируется и становится зелёным
- После смерти: возрождение у чекпоинта, а не в начале
- Не сбрасывается при переходе к следующей жизни

### 40. Выбор сложности
- Экран выбора перед Level Select: Easy / Normal / Hard
- Easy: скорость врагов ×0.7, 5 жизней, двойной бонус времени
- Normal: без изменений (текущий баланс)
- Hard: скорость ×1.3, 2 жизни, половина бонуса времени
- Сохраняется в localStorage

### 41. Летающий Марио
- Новый тип врага type='flying' — летит горизонтально без гравитации
- Движется на фиксированной высоте, не стоит на платформах
- Отталкивается от краёв экрана и от платформ
- Убить можно только прыжком сверху (стомп)
- Значок ✈ над головой + крылья
- Появляется на уровнях 5-9

### 42. Рекорд времени уровня
- Для каждого уровня сохраняется лучшее время прохождения в localStorage
- На экране Level Complete показывает затраченное время
- Если поставлен новый рекорд — анимированное сообщение «⏱ РЕКОРД!»
- Рекорд сравнивается с текущим временем завершения уровня

### 43. Power-up Заморозка
- Голубой ледяной кристалл на уровнях 4-9
- При подборе: все враги замораживаются на 4 секунды (freezeTimer)
- Замороженные враги: синеватый тинт, полная остановка, ещё можно убить стомпом
- Ледяные частицы при активации и истечении действия
- Таймер заморозки в HUD

### 44. Прогресс-бар врагов
- Тонкая горизонтальная полоска вверху экрана под score
- Заполнена полностью в начале уровня, уменьшается при убийстве врагов
- Цвет: зелёный (много врагов) → жёлтый → красный (мало осталось)
- При последнем враге мигает и слегка пульсирует
- Скрыта на боссовом уровне (там есть HP-бар босса)
