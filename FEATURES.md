# Mushroom's Revenge — План фич

## Выполненные фичи

- [x] Мобильное управление (тач-контроллер с виртуальными кнопками)
- [x] Адаптивный canvas (масштабирование под размер экрана)
- [x] Монеты (собираемые предметы на уровнях, +50 очков)
- [x] Двойной прыжок (power-up, появляется на уровнях 3-5)
- [x] Сохранение рекорда в localStorage
- [x] Движущиеся платформы (горизонтальные и вертикальные, уровни 2-5)
- [x] Анимированный фон (движение облаков)
- [x] Система комбо (множитель очков x1-x5 за последовательные прыжки на Марио)
- [x] Эффект частиц при сборе монет (золотые искры)
- [x] Фоновая музыка (чиптюн-мелодия через Web Audio API)

## Запланированные фичи

- [ ] Экран выбора уровня (с разблокировкой по прохождению)
- [ ] Звёзды за прохождение уровня (1-3 звезды в зависимости от очков)
- [ ] Враг-босс на каждом 5-м уровне (большой Марио с повышенным HP)
- [ ] Dash-рывок (по двойному нажатию влево/вправо)
- [ ] Анимация смерти врагов (разлетающиеся пиксели)
